# p2p
1. ```git clone https://github.com/flutter/flutter.git -b stable```
2. ```echo 'export PATH="$PATH:/workspaces/p2p/flutter/bin"' >> ~/.bashrc```
3. ```source ~/.bashrc```
4. ```flutter analyze lib```
5. ```flutter run -d web-server --web-port 8080 --web-hostname 0.0.0.0```
6. ```dart run build_runner build --delete-conflicting-outputs```
